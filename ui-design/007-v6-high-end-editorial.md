# 007 - V6 顶级极简与国际化杂志排版 (High-End Editorial & Swiss Design)

## 1. 深度复盘：为什么 V5 显得“像小作坊”？
用户给出了非常精准的反馈：“风格还不错（杂志/极简的大方向是对的），但是颜色奇怪，布局/字体大小不够，感觉像草台班子/敷衍的产品”。

这是一个非常经典的设计问题：**极简主义 (Minimalism) 和 简陋 (Amateurish) 之间只有一线之隔。**
V5 失败的原因在于：
1. **颜色不够纯粹**：使用了带灰调的“原木色”和“鼠尾草绿”，在没有顶级摄影图配合的情况下，很容易显得“脏”和“旧”，就像发黄的旧纸，而不是高级的杂志。
2. **排版比例失调 (Typography Scale)**：去掉了阴影和卡片后，视觉层级完全依赖字体大小和留白。如果字号不够极端（标题不够大、标签不够精致），就会像是一个没有写 CSS 的裸 HTML 页面。
3. **缺乏结构骨架 (Grid & Structure)**：太多的留白却没有严谨的网格和参考线，会导致内容“飘”在屏幕上，显得松散、敷衍。

## 2. V6 核心重构法则：瑞士极简与国际化排版 (Swiss Design)

我们要打造的是类似于 **Kinfolk、Cereal 杂志、或是顶级奢侈品官网（如 Celine, Apple 极简风格）**的视觉体验。

### 2.1 色彩：极端的克制 (Monochrome)
- **绝对的黑白**：废除一切彩色背景和装饰色。
- **背景 (`--bg-color`)**：`#FFFFFF` (纯白)。没有任何杂质的白，让食物照片成为唯一的色彩来源。
- **主文本 (`--text-main`)**：`#000000` (纯黑) 或 `#111111`。提供最顶级的对比度和锐利度。
- **辅助线 (`--border-color`)**：`#E5E5E5` (非常干净的浅灰)，只用于划分网格。

### 2.2 排版：极致的对比与秩序 (Extreme Typographic Hierarchy)
- **双字体系统**：
  - **大标题 (H1/H2)**：坚持使用 **Serif (衬线体/宋体/明朝体)**。字号要**极其夸张（如 40px - 56px）**，字重用 Normal，展示优雅的笔画细节。
  - **标签、副标题、元数据 (Meta/Tags)**：使用 **Sans-Serif (无衬线体)**。全大写 (UPPERCASE)，字号要**极其小且精致（11px - 12px）**，并增加极宽的字间距 (`letter-spacing: 2px`)。这种强烈的“大与小”的对比，是消除“敷衍感”、建立“高级感”的核心秘诀。
  - **正文**：15px - 16px，行高 1.8，深灰色 (`#666`)，提供完美的阅读体验。

### 2.3 形态与结构：几何与网格 (Geometry & Grids)
- **零圆角 (Zero Border-Radius)**：废除所有的柔和圆角。所有的图片、按钮全部采用 `0px` 锐角。展现出一种刀刻斧凿般的锋利感和印刷质感。
- **绝对对齐 (Alignment)**：利用极其克制但清晰的 1px 浅灰色线条（如表格、分割线）来构建页面的骨架。
- **高级控件**：按钮不再是简单的文字链接，而是**纯黑色的实心矩形** (`background: #000; color: #FFF;`)，带有尖锐的直角。这能立刻提升页面的力量感和专业度。

## 3. 执行细节：把 UI 变成艺术品
1. 所有的边框、卡片阴影全部抹除。
2. 按钮全部改为直角黑底白字。
3. 标签 (Tags) 变成极其精致的细线边框或纯文本加宽间距。
4. 菜谱卡片的图片变成完美的直角矩形，文字与图片严格左对齐。
