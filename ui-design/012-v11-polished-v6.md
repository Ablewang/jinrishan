# 012 - V11 纯粹高定杂志与发丝网格 (Polished Editorial & Hairline Grid)

## 1. 深度共识：找回那个“最好但太空”的版本

您提到的那个“简约化挺好，但是太空了”的版本，正是 **V6 (High-End Editorial / 高端杂志排版)**。

**V6 的核心特征是：**
- 绝对的黑白灰 (纯白背景，纯黑文字)
- `0px` 绝对直角 (没有圆角，极其锋利)
- 强烈的字体对比 (Serif 衬线大标题 + Sans-serif 无衬线小字)
- 没有任何卡片背景色和阴影

**为什么 V6 会觉得“太空”？**
因为去掉了所有的卡片、阴影和背景色后，元素失去了“容器”，只能漂浮在白色的屏幕上。如果没有强大的“骨架”去锚定它们，视觉上就会觉得漏气、散漫、太空洞。

## 2. 打磨方案：如何解决“空”，且不破坏“极简”？

在顶级的实体印刷杂志（如 Kinfolk, Cereal）中，页面也是纯白的，为什么它们不觉得空？
答案是：**严谨的排版留白 (Rhythm) 与 发丝级网格线 (Hairline Grids)**。

这一次，我们**完全回到 V6 的零圆角、纯黑白、衬线体**，但是进行骨灰级的打磨：

### 2.1 发丝级分割线 (Hairline Dividers)
不再使用任何背景色来区分区块。我们引入极细、极淡的 1px 分割线 (`#EAEAEA`)。
- 在每个菜谱之间、每个设置项之间、每个成分之间，使用这条极细的线。
- 线条能提供强大的“视觉锚点”，在不增加任何视觉负担的情况下，瞬间收束住飘散的留白，彻底解决“空洞感”。

### 2.2 极致的字体层级 (Extreme Typography Scale)
- **大字更大，小字更小**。
- 大标题使用 `Playfair Display`，展示优雅的墨迹感。
- 标签 (Tags)、时间、元数据使用极小的全大写字母 (11px, UPPERCASE)，并增加字间距 (`letter-spacing: 1px`)。这种“微缩的精致感”是高端品牌不可或缺的。

### 2.3 建筑般的区块感 (Architectural Blocks)
- 按钮统一为纯黑色的锋利矩形 (`background: #000`)。
- 没有阴影，没有渐变。按下去的反馈是纯粹的透明度变化，展现冷峻的克制感。
- 图片与文字严格左对齐，形成坚实的垂直参考线。

我们现在就去实现这个**“不空的极致杂志风”**。
