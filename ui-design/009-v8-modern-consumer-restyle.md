# 009 - V8 顶级国民级消费应用风 (Modern Consumer / Airbnb Aesthetic)

## 1. 痛点反思：为什么 V7 会得到“丑！”的评价？
用户直接反馈“丑！”，这说明 V7 的**粗野主义 (Brutalism)**、**线框网格 (Wireframe)** 和 **硬阴影 (Hard Shadows)** 完全脱离了大众对“美食应用”的审美预期。
- 粗黑边框和生硬的直角显得过于硬朗、死板，甚至是具有攻击性。
- 美食应用需要传达的是**“美味、温馨、现代、友好”**。
- 用户之前觉得“空”，是因为在极简风下，页面缺乏层级支撑。我们用极其硬核的线条去填补“空”，导致了严重的过度设计，变得“丑陋”。

## 2. 最终解决方案：对标全球顶级国民应用 (如 Airbnb, 小红书, Pinterest)
我们要抛弃“高冷”、“性冷淡”和“硬核排版”，回归**最让人舒服的现代消费级 App 审美**。这种风格的包容度最广，99% 的用户都会觉得“好看、高级且舒适”。

### 2.1 色彩：温暖、食欲、呼吸感
- **品牌主色 (Primary)**：`#FF385C` (Airbnb 经典的珊瑚红/树莓红)。这是全球公认的最具食欲、活力和现代感的主色调。
- **背景底色 (Background)**：`#F7F7F9` (极浅的柔和冷灰)。
- **卡片/表面 (Surface)**：`#FFFFFF` (纯白)。
- **文字**：放弃刺眼的纯黑，使用 `#222222` 作为主标题，`#717171` 作为正文，极其护眼。

### 2.2 形态与层级：用卡片解决“空洞”，用圆角传递“友好”
- **完美圆角 (Squircle)**：卡片使用 `16px` 甚至 `24px` 的大圆角，按钮使用 `12px` 或是全圆角 (`999px`)。展现出极致的亲和力。
- **柔和弥散阴影 (Soft Diffused Shadows)**：
  - 彻底删掉 4px 纯黑硬阴影。
  - 使用大范围、极低透明度的阴影：`0 12px 32px rgba(0, 0, 0, 0.08)`。这会让白色的卡片像云朵一样优雅地浮在浅灰色的背景上，既解决了“空”的问题（因为有了卡片的实体），又保证了页面的通透感。

### 2.3 排版与留白 (Typography & Spacing)
- 放弃生硬的衬线体 (Serif)，全站统一使用无衬线体 (Inter / PingFang)，加粗标题 (`font-weight: 700`)，带来干净利落的现代感。
- 组件之间保持充足的 Padding（如 24px），让内容能够自由呼吸。

## 3. 核心页面重构指南
- **Home 页**：背景是浅灰，推荐菜谱是一张极其精美的纯白圆角卡片，带有极其柔和的阴影。图片占满卡片上半部分，下方是干净的排版。
- **RecipeDetail 详情页**：同样采用干净的白色卡片结构包裹核心内容。
- **WeeklyPlan / Settings 等**：用纯白圆角卡片将不同的模块进行分组包裹（Grouping），页面天然产生骨架感，再也不会“空”了。
