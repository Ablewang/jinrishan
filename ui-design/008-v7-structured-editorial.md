# 008 - V7 结构化国际高定排版 (Structured Editorial & Grid Wireframe)

## 1. 深度诊断：为什么“有品牌感了，但感觉有点空”？

用户反馈 V6 已经具备了品牌的简约感，但“有点空”。
这是一个在极简设计中非常经典的问题。当我们去掉了阴影、圆角和多余的色彩后，页面的留白（White Space）就成为了绝对的主角。

如果这些留白没有**“明确的边界”**和**“内在的张力”**，用户的视觉就会缺乏锚点，感觉内容是在屏幕上“漂浮”或“漏气”的，这就是“空”的来源。

## 2. V7 破局核心：用网格线锚定视觉 (The Visible Grid System)

为了保留品牌级的高级感，同时消除“空洞感”，我们要引入经典的**瑞士网格排版系统 (Swiss Grid System)**，并将其**显性化 (Wireframe Aesthetic)**。

### 2.1 显性化的线条结构 (Visible Borders)
- 将之前隐形的、微弱的浅灰线条，升级为**明确的结构线**。
- 采用深色的 1px 边框（`border: 1px solid #000` 或强对比的浅灰）将内容严格地“框”起来。
- 菜谱卡片不再是无边界的漂浮物，而是带有明确黑色边框的“信息模块 (Data Block)”。
- 图片下方、标题下方增加实线分割，让页面像是一份**严谨排版的报纸 (Newspaper) 或图鉴**。留白一旦被线条框住，就不再是“空洞”，而是“有张力的呼吸空间”。

### 2.2 微小而克制的点缀色 (Micro-Accents)
- 纯黑白容易产生视觉疲劳。我们引入一抹极其克制、深邃的**“朱砂红” (`#D32F2F`)**。
- 这抹红色**绝对不用于大面积的按钮或背景**，仅仅作为：
  - 极细的导航指示线 (Indicator line)
  - 关键步骤序号的颜色
  - 未读、或某个精细的 Icon 颜色
  - 这种“万黑丛中一点红”的做法，是顶级出版物常用的手法，能瞬间拉升页面的精神气。

### 2.3 底层与表层的对比 (Background vs Surface)
- 背景色 (`--bg-color`) 采用极其微弱的冷灰/浅纸白 (`#F7F7F7`)。
- 主体卡片区 (`--surface-solid`) 使用纯白 (`#FFFFFF`)。
- 虽然只有 3% 的色差，但配合黑色的边框线，能让内容区产生扎实的存在感（Grounded），不再像之前那样与背景融为一体、无边无际。

## 3. 具体执行层面

- **Home 推荐页**：
  - 将整个卡片加上 `border: 1px solid var(--text-main);`，底部加上厚重的阴影（硬阴影 `box-shadow: 4px 4px 0 #000`，带来粗野主义 Brutalism 的厚实感，解决“空”的问题）。
  - 图片与文字之间用 `border-bottom: 1px solid #000` 隔开。
- **RecipeDetail 详情页**：
  - 食材列表改为严格的表格风 (Table Row)，带有上下黑线。
  - 步骤列表的连线加深，形成时间轴。
- **按钮控件**：
  - 引入硬底阴影 (`box-shadow: 4px 4px 0 #000`)，当按下去时产生物理位移。这既符合当前的锐利直角风格，又极大增加了页面的“重量感（Density）”，彻底解决漂浮和空洞。
