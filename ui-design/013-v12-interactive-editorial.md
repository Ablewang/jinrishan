# 013 - V12 交互式高定与温润排版 (Interactive Editorial & Warm Paper)

## 1. 深度复盘：风格正确，但体验与细节欠缺

用户对当前的“极简杂志风 (Editorial)”大方向是认可的，但提出了四个极其关键的优化点：
1. **交互太重**：看做法需要跳页面，返回太麻烦，移动端也没有滑动返回体验。这是极大的体验破坏。
2. **配色太生硬**：绝对的 `#FFFFFF` 和 `#000000` 虽然高级，但在数字屏幕上略显刺眼，缺乏温度。
3. **内容显得空**：卡片右侧虽然有极简感，但信息密度不够，显得不够丰富。
4. **排版比例**：PC端图片过大，显得有些失衡。

## 2. V12 核心重构：从“平面杂志”到“交互式读物”

### 2.1 革命性的交互：Sliding Drawer (抽屉式面板)
- 我们不再让用户跳转到 `/recipe/:id` 新页面去查看做法。
- 引入 **抽屉组件 (Drawer)**：
  - **PC 端**：从屏幕右侧优雅滑出，宽度固定，背景带有浅浅的遮罩层。用户可以随时点击遮罩层关闭，立刻回到推荐页/周计划。
  - **移动端**：从屏幕底部滑出 (Bottom Sheet)，占据 90vh，顶部带有下滑指示条。极其符合现代 App 的单手操作直觉。
- 所有的详情查看都在“当前页”完成，彻底解决“切换太麻烦”的痛点。

### 2.2 温润的纸质配色 (Warm Paper Palette)
- 放弃刺眼的纯白背景。全局背景 `--bg-color` 改为极其温润的**“燕麦白 / 宣纸色” (`#FAFAF8`)**。
- 内容卡片 `--surface-solid` 保持纯白 (`#FFFFFF`)。这 2% 的色差，既保留了极简感，又增加了页面的温度和层次。
- 放弃纯黑文字。改为深邃的**“石墨灰” (`#1A1A1A`)**，大幅降低视觉疲劳。
- 加入一抹**“焦糖色 / 陶土红” (`#A67A5B`)** 或 **优雅的深棕** 作为点缀（例如标签、高亮元素），让页面更有食欲和生活气息。

### 2.3 丰满的排版与信息密度
- **重塑卡片比例**：PC 端让图片和文字的比例变为 `1:1` 或文字区更大，让排版更加均衡。
- **增加信息元 (Metadata)**：在卡片标题下方，直接抽出菜谱的核心信息（如卡路里、制作时间、难度），用精致的 Icon + 细小文本展示，提升内容丰富度。
- **重新调整间距**：优化 Tab 栏的间距，让呼吸感更加均匀。

这次的升级，将让这个极简设计真正具备“现代 App”的流畅度和“顶级出版物”的温度。
